Big-O gösterimi: O(n^2)

## 3
Time Complexity: 1)Average case: Aradığımız sayının ortada olması.
2)Worst case: Aradığımız sayının sonda olması. 
3)Best case: Aradığımız sayının dizinin en başında olması.

## 4
